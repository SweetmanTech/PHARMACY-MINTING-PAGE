## Wura, A Narrative UI

- Single by J. Mack ENT.

Mint Page: [planetsun.xyz](https://planetsun.xyz/)

### Get Started

1. install dependencies with `yarn`
2. update ENV variables in `.env.local`
3. start the server with `yarn dev`
